# Kivy Calculator
A simple calculator application built using Python and the Kivy framework.  
This project demonstrates GUI development, layout management, and event handling in Kivy.

## Features
- Numeric keypad
- Basic arithmetic operations
- Clear button
- Result evaluation

## Requirements
- Python
- Kivy

## How to Run
1. Install Kivy
   pip install kivy

2. Run the application
   python main.py

## Project Purpose
This project is created for learning GUI application development using Python Kivy.
